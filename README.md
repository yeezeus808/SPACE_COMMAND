# cosmic_pong-
repository for all files for cosmic pong 

STEP 1 _ Install VS Code
1. Go to https://code.visualstudio.com/download
2. Click Download for Windows (or Mac/Linux)
3. Run the installer and follow the prompts  all default settings are fine

STEP 2 _ Install Python
1. Go to https://www.python.org/downloads/
2. Click Download Python (get the latest version)
3. Run the installer  make sure to check "Add Python to PAT” before clicking Install
4. Click Install Now

STEP 3 _ Set up VS Code for Python
1. Open VS Code
2. Click the Extensions icon on the left sidebar (or press Ctrl+Shift+X)
3. Search Python and install the one made by Microsoft

STEP 4 _ Install Pygame
1. In VS Code, open the terminal 
2. Type this and hit Enter:
pip install pygame-ce

STEP 5_ Run the game
1. Open the game folder in VS Code via File → Open Folder
2. Open   game_error_v14.py
3. run it from the terminal
